Escape Room

This project is made using Unity which showcases:





Interaction

Clicking on the buttons triggers the animation and checking of the puzzles.

Timer

Triggers the UI distortion effect and alarm sound based on the time left.

Scenes

Whenever the player wins or loses the game, it leads to a different scene.







C#

The programming language used for the functionality of the interactions, timer, and changing of scenes.
